Installationsanleitung

  1. Installiere cmake
  2. Installiere opencv
  3. Installiere mmal (meistens bereits in der Raspi Firmware enthalten [raspberrypi-firmware-tools])
  4. Buildprozess servoblaster
    1. git clone https://github.com/richardghirst/PiBits
    2. cd PiBits/ServoBlaster/user
    3. make
    4. Optional auf dem raspbian OS (triggert Autostart): make install
  5. Buildprozess raspicam
    1. cd <mechanical_eyes dir>/libs/raspicam-0.0.5/
    2. mkdir build && cd build
    3. cmake ..
    4. make && make install
  6. Buildprozess mechanical_eyes
    1. cd <mechanical_eyes dir>
    2. mkdir build && cd build
    3. cmake ..
    4. make && make install
